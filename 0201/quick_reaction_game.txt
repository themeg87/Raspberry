
===순발력 게임===

led가 10회 깜빡이는 동안 각 색에 대응하는 버튼을 누른다
* 빨강 -> SW2, 초록 -> SW3, 파랑 -> SW4
  빨간 led가 켜지면 SW2 버튼을 누르는 식에 순발력 게임 

조건1 : 색은 랜덤으로 출력된다
조건2 : 불이 들어와 있는 시간도 랜덤으로 선택된다(단, 0.5~1.0 사이에서 랜덤)

판정1 : 버튼을 올바르게 눌렀는가?
불이 들어와있는 동안 해당 색에 대응하는 버튼을 올바르게 눌렀는지 확인
다른 버튼을 눌렀다면 "bad"가 출력된다

판정2 : 버튼을 시간내에 눌렀는가?
불이 들오와있는 시간동안 버튼을 눌렀는가
아무 버튼도 누르지 않았다면 "miss"

판정3 : 불이 들어와 있는 동안 색에 대응하는 버튼을 눌렀다면 "perfect" 출력

10회 반복한후 perfect가 출력 된 횟수에 따라 점수를 출력 한다
ex) "최종 점수는 8점"
